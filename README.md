# Synchrono — Landing Page

A modern, animated landing page, built with React 19, TypeScript, Vite, and Tailwind CSS v4. Create to practice using Framer Motion. 

Note: this landing page is not responsive.

## Tech Stack

| Tool | Purpose |
|---|---|
| React 19 | UI framework |
| TypeScript | Type safety |
| Vite | Build tool & dev server |
| Tailwind CSS v4 | Styling |
| Framer Motion | Animations & scroll effects |
| Matter.js | Physics-based shape grid |
| OGL | WebGL rendering |
| Lucide React | Icons |
| shadcn/ui | Base UI components |
